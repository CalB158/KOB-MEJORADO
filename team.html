¿Quiénes somos?

Somo un grupo de ingenieros con el fin de hacer esta página web dinámica, de compra/venta donde se puede adquirir u ofrecer artículos de segunda mano como inmuebles, aparatos electrónicos, videojuegos, celulares, computadoras, vehículos, etc. ¡Qué te diviertas!.

Funciones

- El sistema solo podrá ser utilizado en el departamento de cómputo KOB ya que el tanto análisis como diseño corresponde a este.

- La velocidad de transferencia de los datos dependerá de la velocidad conexión a Internet, que no tenga embotellamientos y dado el caso que no haya conexión a Internet se suspenderá el uso del sistema.

- El sistema solo podrá ser utilizado por personal correspondiente al departamento esto con la finalidad de evitar malos usos

Características:

Los usuarios deberán registrarse en el sistema para así obtener una mejor experiencia.

Al registrarse, el usuario debe proporcionará sus datos como: Nombre, Apellido, E-mail, Usuario, Password, Fecha de nacimiento y Número telefónico

El sistema ofrecerá al usuario información general acerca de las categorías de los productos que se ofrecen.

El sistema ofrecerá una lista de las categorías como de casa y jardín, entretenimiento, electrónica, entre otros.

Todo esto con el bjetivo que sea atractiva y cuente con diferentes secciones para que los usuarios puedan encontrar el artículo deseado con rapidez, manteniendo el orden y el respeto entre los usuarios

Conclusión

La interfaz consistirá en un conjunto de ventanas con botones, listas y campos de textos. Ésta deberá ser construida específicamente para el sistema propuesto y, será visualizada desde un navegador de internet.
